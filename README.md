# 構造分析ポートフォリオ

本リポジトリは、筆者が行った構造分析に基づくレポート作品を体系的にまとめたポートフォリオです。扱うテーマは異なりますが、いずれの作品も、現象を構造として捉え、因果関係、情報密度、主体配置、制度設計といった観点から分析を行うという共通の方法論に基づいています。個別のテーマに依存しない「構造そのものの理解」を目的としており、各レポートは独立した作品でありながら、全体として一つの分析体系を形成しています。

## 作品一覧

### 日本企業のDXはなぜ「合理的に」停滞するのか

フォルダ：`dx-report/` 

本レポートでは、日本企業のDXが文化論や精神論ではなく、組織構造の合理性によって停滞するという視点から分析を行っています。意思決定の階層構造、責任の配置、リスク回避の制度設計、情報流通の分断など、複数の構造要因がどのように相互作用し、結果として停滞を生み出すのかを整理しています。レポート本文に加えて、スライド、Q&A、ロードマップなどの補助資料を併置し、分析の背景と展開を多面的に参照できる構成としています。



### なぜ「視聴者を無視する配信」が最高の没入を生むのか

フォルダ：`immersion-report/` 

本レポートでは、ゲーム配信における没入の源泉を、配信者の個性やコミュニケーションではなく、ゲームが持つ情報密度と不可逆性に置く視点から分析しています。主体の置換、情報流の一方向性、ノイズの排除、観測者の位置づけといった構造的要素が、なぜ「視聴者を無視する」という振る舞いを通じて最大化されるのかを整理しています。本作品はレポート本文のみで構成されており、単体で分析の全体像を把握できるように設計しています。


### **身体操作の再定義における「報酬とコスト」の不整合**

##### **フォルダ：** `motor-cognition-report/`

本レポートでは、分割キーボードの代替案として試みた「片手入力」が、なぜ合理的な意図に反して実務に定着しなかったのかを分析しています。長年の反復で形成された「高速道路」のような既存スキルが新習慣を拒絶する**「熟練者の罠」**や、動作の文脈（コンテキスト）の不一致、さらには**「長期的な健康報酬」が「短期的な生産性コスト」に敗北する報酬構造**など、身体と認知の多層的な不整合を解明しています。道具の置換ではなく「動作の再構成」が必要であるという結論を導き出した、自己の認知プロセスに対する構造分析です。



## 読み方について

各フォルダには、テーマごとのレポート本文を配置しています。作品によっては、スライドやQ&A、ロードマップなどの補助資料を併置しているものもありますが、補助資料の有無にかかわらず、本文を中心に読み進めることで分析の全体像を把握できる構成としています。本ページで作品群の概要を確認した上で、興味のあるテーマのフォルダへ進むことで、個別の分析内容を体系的に参照できます。
