# **構造分析ポートフォリオ**

本リポジトリは、複雑な現象を構造として捉え、因果関係・主体配置・制度的制約・情報流の特性といった観点から整理した分析レポートをまとめたものです。扱うテーマは異なりますが、すべての作品で共通の分析枠組みを用いており、テーマ依存ではなく「構造そのもの」を理解することを目的としています。

本リポジトリで用いている構造化手法は、調査レポート、業務手順書、Q&A、スライド資料など、複数の文書形式に応用可能なものです。異なる領域に対しても同一の方法論を適用できる点を示すため、テーマの幅を意図的に広く設定しています。

## **分析方法論（共通フレーム）**

各レポートは、以下の構造化フレームに基づいて作成しています。

1. **現象の定義**
    
2. **関係主体の整理**
    
3. **制約条件（制度・文化・経済・技術など）**
    
4. **因果構造の可視化**
    
5. **ボトルネックの特定**
    
6. **構造的に解消されない理由**
    
7. **仮説（構造的視点からの結論）**
    

このフレームにより、異なるテーマでも再現性のある分析が可能です。

## **作品一覧**

### **1. 日本企業のDXはなぜ「合理的に」停滞するのか**

**フォルダ：** `dx-report/`

日本企業のDXが文化論ではなく、組織構造の合理性によって停滞するという視点から分析しています。意思決定の階層構造、責任の配置、リスク回避を前提とした制度設計、情報流通の分断など、複数の構造要因がどのように相互作用し停滞を生むのかを整理しています。レポート本文に加えて、スライド、Q&A、ロードマップなどの補助資料を併置し、分析の背景を多面的に参照できる構成としています。

### **2. なぜ「視聴者を無視する配信」が最高の没入を生むのか**

**フォルダ：** `immersion-report/`

ゲーム配信における没入の源泉を、配信者の個性ではなく、ゲームが持つ情報密度と不可逆性に置く視点から分析しています。主体の置換、情報流の一方向性、ノイズの排除、観測者の位置づけといった構造的要素が、なぜ「視聴者を無視する」という振る舞いを通じて最大化されるのかを整理しています。本作品はレポート本文のみで構成され、単体で分析の全体像を把握できるように設計しています。

### **3. 身体操作の再定義における「報酬とコスト」の不整合**

**フォルダ：** `motor-cognition-report/`

片手入力の導入が合理的意図に反して定着しなかった理由を、熟練者の罠、動作文脈の不一致、長期的な健康報酬と短期的な生産性コストの非対称性といった観点から分析しています。長年の反復によって形成された既存スキルが新習慣を拒絶する構造や、身体と認知の多層的な不整合を扱った自己分析レポートです。

## **読み方について**

各フォルダにはテーマごとのレポート本文を配置しています。作品によっては補助資料を併置しているものもありますが、補助資料の有無にかかわらず、本文を中心に読み進めることで分析の全体像を把握できます。本ページで概要を確認した上で、関心のあるテーマのフォルダへ進むことで、個別の分析内容を体系的に参照できます。
