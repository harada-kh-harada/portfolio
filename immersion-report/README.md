# なぜ「視聴者を無視する配信」が最高の没入を生むのか


本レポートは、ゲーム実況における没入体験を、個人の感性や配信者の技量といった不確実な要素ではなく、ゲーム・配信者・視聴者の三者が形成する構造として捉え直すことを目的としている。一般には、視聴者への挨拶やコメントへの反応といった対人的なやり取りこそが「良い配信」をつくると考えられているが、筆者が強い没入を感じた実況体験を振り返ると、この前提は必ずしも成立しない。むしろ、配信者が視聴者に向き合うほど、意識がゲームの外側へ押し戻され、没入が途切れる感覚が生じていた。

この没入の差異は、配信者の性格や努力では説明できない。没入は、ゲーム密度、注意拘束、対人構造強度、主体置換度という四つの変数が連鎖することで生まれる構造的現象である。本レポートでは、これらの変数がどのように作用し、どのような条件下で視聴者が「観客」から「観測者」へと変化するのかを明らかにする。特に、麻雀実況のような高密度・不可逆ゲームでは、配信者の注意がゲームに拘束され、対人構造が自然に弱まり、視聴者が配信者と同じ情報の流れに巻き込まれる主体へと置換される。この構造こそが、強い没入を生む根本要因である。

本レポートは、没入の価値観の優劣を論じるものではない。配信者との会話や親密さを中心に据えた配信に没入する視聴者も存在し、没入の源泉は一様ではない。しかし、筆者が強く没入を感じた「ゲームが主体となる構造」は、四変数の因果連鎖によって再現可能に説明できる。本レポートは、この構造を理論化し、没入が偶然の体験ではなく、観測可能で分析可能な現象であることを示すものである。
