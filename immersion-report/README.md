# なぜ「視聴者を無視する配信」が最高の没入を生むのか

本フォルダには、構造分析レポート「なぜ『視聴者を無視する配信』が最高の没入を生むのか」の本文を収録しています。本レポートは、ゲーム配信における没入の源泉を、配信者の個性やコミュニケーションではなく、ゲームが持つ情報密度と不可逆性に置く視点から分析しています。配信者と視聴者の関係性を中心に据える従来の理解とは異なり、主体の置換、情報流の一方向性、ノイズの排除といった構造的要素がどのように没入を形成するのかを整理しています。

本作品はレポート本文のみで構成されており、補助資料を必要とせず単体で分析の全体像を把握できるように設計しています。まずは本文を通読していただくことで、没入がどのような構造的条件によって成立するのかを理解できる構成としています。
